# Data Analytics Coursework Projects

This repository contains a collection of academic and practical projects I have completed as part of my coursework and personal learning in Data Analytics, Data Engineering, Probability, Programming, and Data Visualization.

Each project is organized in its own folder and includes all relevant files such as notebooks, reports, datasets, and presentations. The projects demonstrate skills in:

- Data preprocessing and exploratory analysis  
- Machine learning model development and evaluation  
- Statistical reasoning and probability analysis  
- System and application development  
- Data visualization and storytelling  

## Projects Included

1. **DiabetesRiskAnalysisProject** – Predicting diabetes risk using machine learning.  
2. **Airlines Reviews Sentiment Analysis** – Analyzing customer reviews and sentiment patterns.  
3. **Deepfakes Detection System** – Detecting deepfakes using neural networks.  
4. **Environment Data Visualization** – Visualizing environmental data trends.  
5. **KIFAAppDevelopment** – App development project aligned with SDG goals.  
6. **ThreatsofAIinFintechSector** – Research on AI-related risks in the fintech sector.
